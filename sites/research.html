Peter Nicholas Krämer
PhD Student Machine Learning // University of Tübingen

I am working on probabilistic numerical algorithms for differential equations, and related topics. Machine learning applications are always welcome. Most of the methods I come up with in my papers end up on Github (for example as part of ProbNum , but I write other code, too).

At the Institute for Numerical Simulation in Bonn, I have been working on numerical linear algebra for large linear systems involving kernel matrices, and scattered data approximation. On the media page, you can find an abstract of my MSc thesis.

If you would like to have a chat, please feel invited to get in touch!
